### 函数类型

> 为了使用接口 表示函数类型，我们需要给接口定义一个**调用签名**。 它就像是一个**只有参数列表和返回值类型的函数定。参数列表里的每个参数都需要名字和类型。**

### 索引类型

索引只支持两种类型，一种是number， 一种是string，但实际上在js转换后，他们都是指向同一个属性，所以要注意不能用“相似”的值来声明不同的返回值类型，**要保持一致性**；
