# 媒体查询

媒体查询能用在很多地方。
```
<link media="print and (color)">
<style media="">
  @import url() all
</style>
```

#### 基础媒体查询

**all**
所有媒体

**print**
打印

**screen**
呈现文档的屏幕媒体。


#### 复杂的媒体查询

引入逻辑运算符

**and**
类似于JS里的 &&。

**not**
*对整个查询的结果取反*，如果所有条件都满足，不应用样式表。只能用在媒体查询的开头

**only**
目的就是向后兼容，因为老得浏览器不认识它就自动过滤了这条样式。

> 媒体查询不存在or，但是 可以使用逗号(,) 分隔媒体查询即可起到or的含义。
